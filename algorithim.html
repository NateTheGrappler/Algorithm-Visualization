How To Use

The top buttons open up several helpful windows that tell you more about this website or array. In order they open the: Help Window (This very window), Description Window (says how algorithm works), Code Window (Showcases code in three major languages), and the Input Window, which allows for inputting your own array to sort.

These input fields control the array that you see in the middle of the screen. The min and max values are the given range of your array. The number of elements input is then the size of the array, with all values specified within the range. Clicking the generate array button will generate a random array with the values you outlined.

These buttons control the visualization of sorting with the array. They are fairly self explanatory, the start and stop buttons control if the sorting is happening. While pause will, pause, the sorting. With reset reverting back to the unsorted array.

The image slider dedicates the speed of the visualization. If the slider is set to max, it will sort the array without the visualization to maximize speed. While pausing the array if the speed is set to zero.

This is the array that represents you array, it will change colors as it is sorting and tell you once it is fully sorted. Happy sorting!
